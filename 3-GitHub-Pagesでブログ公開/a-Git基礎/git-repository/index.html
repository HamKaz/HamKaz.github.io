事前作業

Generate SSHKey

ローカル側でSSHKey作成

Register SSH Key

GitHub側でSSH Key登録

Confirm SSH Access

ローカル側でアクセス確認

ローカルからリモートにプッシュ（一回目）

cd

アップロード対象となるディレクトリに移動

git init

gitのセットアップ -> .gitフォルダが作成される

git add

変更履歴を保存

git commit

変更履歴を確定

[Create New Repository]

GitHub側で新規リポ作成

git remote add origin

リモートのブランチ設定

git push origin master

ローカルからリモートにプッシュ（アップロード）。ローカルのブランチとリモートのブランチを合わせる（今回はmaster）

ローカルからリモートにプッシュ（２回め以降）

git add

git commit

git push
