    • アプリ開発までの道のり

    Android Studioでのスマホアプリ開発は、環境構築やアプリ開発が難しいものでした。近年、アプリケーションをOSを問わないクロスプラットフォーム環境が開発され、その中で比較的簡単に環境が構築でき、アプリ開発を体験できるFlutterについて体験した内容をまとめました。

    Android Studioの難しさ

    Android Studio単体でのAndroidスマホのアプリ開発は、初学者である私にはとっつきにくいように感じました。

    React Nativeへの違和感

    最近主流になっているクロスプラットフォーム開発環境のReact Native + Expoで開発テストを行ったところ、LAN経由前提のアプリ実行環境で、QRコードをスマホで読み取って、Expo上でアプリを実行するサーバー・クライアントスタイルということが分かりました。

    ネィティブコードがダウンロードされて実行するのではなく、Expo上でのみ実行されるという点にとても違和感を感じていました。

    Flutterでスマホアプリ

    Flutterは、Googleが開発したモバイルアプリケーション用SDKです。

    Flutter - Beautiful native apps in record time

    Flutter上で開発したアプリケーションは、iOSとAndroidのどちらのプラットフォーム上でも動作させることができ、開発言語はDartとなります。

    テストして感じた点として、Androidのネィティブコードをダウンロード実行するというがとても良いと感じました。

    3) 追加環境の確認

    コマンドFlutter Doctorを実行し、不足している環境を確認します。

    参考サイト：Flutter - Run flutter doctor

    [ x ]が表示されたため、以下のようにして追加設定しました。

    [ v ] Flutter

    flutter doctorコマンドが実行できるということは、Flutterはインストールされています。

    [ ! ] Android toolchain

    必要に応じて、Android Licenseをインストールします。

    [ x ] Android Studio

    Android Studioにflutterとdartのプラグインをインストールするよう要求してきますので、下記の手順でインストール方法しました。

    • Pluginsの検索ダイアログでflutterと打ち込み、検索します。

    • flutterに該当する項目がない場合、下にある（リモート）リポジトリをクリックします。

    • Dartを先にインストールするよう促されるのでYesを押します。

      JetBrainsから、Dartをインストール

      Repositoriesから、Flutterをインストール

    参考サイト：Flutter plugin not installed error;. When running flutter doctor

    [ x ] VS Code

    Visual Studio Codeに、DartとFlutterのExtensions（拡張機能）をインストールします。

    ext install Dart-Code.flutter

    [ x ] Connected device

    エミュレータ上でのテスト、または実機でのテストデバイスを設定する必要があります。

    次の項目では、エミュレーターを設定してみました。

    4) デバイスを設定する

    エミュレーター

    Android Studioのエミュレーターは、AVD managerで設定します。

    Android StudioのメニューからTools > AVD managerを選択し、仮想デバイス（Pixel2などを選択して）作成します。

    参考サイト：Flutterの環境構築

    Android Emulator 上でアプリを実行する

    実機テスト

    Android adb ツールパッケージをインストールします。

    まとめ

    Android StudioとFlutterの組み合わせで、簡単にアプリ作成・ダウンロード・実機テストが出来るようになりました。

    そしてAndroidのネィティブコードをダウンロード実行する点がとても優れており、テストコードといえどもネィティブなので、PC切り離し後もテストできるところが長所として挙げられます。
