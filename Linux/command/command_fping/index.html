LAN上の機器調査

家庭や小規模オフィスのLANでは、ルーターが端末のIPアドレスを管理します。

例えば端末がルーターのDHCP機能を使ってネットワーク接続する場合、ルーターのDHCPサーバーからIPアドレスが振り出されます。

プリンターサーバーやファイルサーバーなど、一定の機能を持つサーバーには固定IPアドレスが割り振られます。

  • ルーター（IPアドレス振り出し）
  • L 端末1（^ 問い合わせ・v IPアドレス取得）
  • L 端末2（^ 問い合わせ・v IPアドレス取得）
  • L 端末3（^ 問い合わせ・v IPアドレス取得）
  • L プリンターサーバー（固定IPアドレス）
  • L ファイルサーバー（固定IPアドレス）

そこで同一LAN上（同一サブネットマスク上）のIPアドレスを検索する方法として、Linuxではfpingコマンドを用います。

fpingコマンドはネットワーク上の機器と疎通確認する方法で、オプション指定をつけて検索すると、同一ネットマスクに接続されている機器を簡単に調べることができます。

fpingコマンド - ネットワークの疎通確認

Linuxにはfpingコマンドが用意されており、ネットワーク上の機器との疎通確認を行うことができます。

インストールに必要なコマンド

apt install

インストール手順

使用方法

サブネットマスクの範囲指定

-g <範囲>

使用例

fping -g 192.168.1.0/24

このようにして、同一サブネット上のIPアドレスを検索することができました。

まとめ

最近はIPアドレスを直接打ち込むこと無く、ローカルネットの名前解決をmDNSで簡単に行う方法が普及し、IPアドレスを忘れてしまうことが多くなりました。

そのため、同一ネットワーク上の機器のIPアドレスを調べる方法を覚えておくと便利だと思いました。
