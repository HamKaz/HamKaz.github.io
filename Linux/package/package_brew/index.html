Homebrewの成り立ち

Linuxのディストリビューションによらないソフトウエア、Windows、Mac OS、Linuxなどのマルチプラットフォームで提供されるソフトウエアは、パッケージ管理apt以外の方法で提供されています。

HomebrewはMac OSのコミュニティで発展したパッケージ管理で、Apple社で提供しないソフトウエアをインストールできます。HomebrewではLinuxのパッケージ管理aptなどで提供されるソフトウエアがインストールできます。

またLinuxでは、Linuxbrewというパッケージ管理として発展し、aptで提供しないソフトウエアをインストールすることができます。

まとめ

brewを用いるとインストールが簡単になります。

またbrew実行時には、apt install実行時のようにroot権限を必要としません。そのため、システムに深く根ざす依存関係が発生しないことがbrewの特徴になります。
