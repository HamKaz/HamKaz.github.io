aptで配布されるソフトウエア

Linux上のソフトウエアはたくさんのコミュニティで開発されており、Debian及びDebian系のコミュニティでは、aptでソフトウエアパッケージを管理します。

参考サイト：Snap形式のパッケージの作り方

ソフトウエアの開発目的や提供するターゲットによっては、aptによる配布方法が不向きであったり、ディストリビューションへの依存関係が少ないインストール方法が望まれる場合があります。

こういった背景からパッケージ管理snapが開発されました。

Snap - Linuxソフトウエアのパッケージ管理

snapはCanonical社が開発したパッケージ管理システムで、ソフトウエアパッケージをディストリビューションや依存関係によらずインストールすることができます。

最近では、MicrosoftのプログラミングエディタであるVisual Studio Code、Googleのスマホアプリ開発環境であるAndroid Studioなども、Snapによりソフトウエア配布されるようになっています。

このサイトを参考に、最新版と安定版のNode.jsの切替を行ってみました。
